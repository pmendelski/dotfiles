# Claude Code - dotfiles

Worth versioning files:
- ~/.claude/settings.json — model, theme, UI preferences, hooks, permissions
- ~/.claude/keybindings.json — custom keyboard shortcuts
- ~/.claude/CLAUDE.md — your personal global instructions to Claude
- ~/.claude/commands/ — custom slash commands you've defined

## Remote notifications

```
- claude-remote - comamnd to start a remote session and send notifications to mobile
- `/remote-on` - Sends notification from the current session to mobile
- `/remote-off` - Turn it off mid-session
```
